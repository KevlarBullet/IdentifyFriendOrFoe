# Identify Friend or Foe
Identify Friend or Foe. A tool for categorising players into groups and prefixing them.

# Explanation
With this, you can press G to open the grouping window, create groups, remove them, and when you select one, you can add or remove selected players. These players will selected will have the group prefixed before their name in the tab list and in their nametag

# Installation

Put this in your mods folder with 1.12.2 forge.
